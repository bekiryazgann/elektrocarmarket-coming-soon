# Elektrocarmarket - Coming Soon Page

A modern, responsive "Coming Soon" landing page for **Elektrocarmarket**, an automotive parts company. This page serves as a professional placeholder while the main website is under development, providing customers with direct contact information and location details.

## üöÄ Features

-   **Professional Branding**: Integration of the official Elektrocarmarket SVG logo as a React component.
-   **Direct Communication**: Integrated WhatsApp button for instant messaging.
-   **Location & Directions**: Google Maps integration for easy navigation to the physical store.
-   **Responsive Design**: Optimized for mobile, tablet, and desktop devices.
-   **Modern UI/UX**: Built with Tailwind CSS 4 for a sleek, high-performance interface.
-   **Smooth Animations**: Entrance animations for a polished user experience.

## üì¶ Getting Started

### Prerequisites

-   [Node.js](https://nodejs.org/) (Latest LTS recommended)
-   [Bun](https://bun.sh/) (Optional, but recommended for faster dependency management)

### Installation

1.  Clone the repository:
    ```bash
    git clone https://github.com/your-username/elektrocarmarket-coming-soon.git
    cd elektrocarmarket-coming-soon
    ```

2.  Install dependencies:
    ```bash
    bun install
    # or
    npm install
    ```

3.  Start the development server:
    ```bash
    bun dev
    # or
    npm run dev
    ```

4.  Build for production:
    ```bash
    bun run build
    # or
    npm run build
    ```
